# Sidecar binaries

`solver/` is the PyInstaller onedir build of the Python solver sidecar. It is
generated by `scripts/build_sidecar.sh`, gitignored, and bundled into the app as
a Tauri resource (`bundle.resources` in `tauri.conf.json`). Rebuild it with
`just package`.

This README is committed only so the `binaries/**/*` resource glob matches at
least one file on a fresh checkout. `tauri-build` errors at compile time if the
glob is empty, which would otherwise break `cargo`, `npm run tauri dev`, and CI
before the sidecar has been built.
